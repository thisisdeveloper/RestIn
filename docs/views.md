# Views Documentation

## Main Views

### CustomerView
- Primary view for customers
- Features:
  - Menu browsing
  - Category filtering
  - Search functionality
  - Cart access
  - Promotional content

### SearchPage
- Restaurant search functionality
- Features:
  - Location-based search
  - Restaurant filtering
  - Quick selection

## Order Management Views

### CartPage
- Dedicated cart management
- Features:
  - Item management
  - Order summary
  - Checkout process

### OrdersPage
- Order history and tracking
- Features:
  - Active orders
  - Order history
  - Status tracking
  - Cancellation options

## Account Management Views

### AccountPage
- Account settings and management
- Features:
  - Profile access
  - Settings
  - Orders history
  - Wallet access

### ProfilePage
- User profile management
- Features:
  - Personal information
  - Preferences
  - Contact details

### AuthPage
- Authentication view
- Features:
  - Login
  - Registration
  - QR code scanning option

### DiscountsPage

### RefundsPage
